Sistema de Venta Java y MySQL

Descripción

Proyecto desarrollado en Java con conexión a base de datos MySQL. El sistema permite gestionar productos, clientes, proveedores y ventas.

Módulos principales

  • Login de usuarios
  • Gestión de productos
  • Gestión de proveedores
  • Gestión de clientes
  • Registro de ventas
  • Base de datos MySQL

CI/CD

El proyecto utiliza GitHub Actions para automatizar la validación de estructura, compilación Java, revisión de base de datos y despliegue básico en GitHub Pages.
